```java
public Page<BookDTO> findBooks(BookQuery query, Pageable pageable) {
    Specification<Book> spec = (root, criteriaQuery, cb) -> {
        List<Predicate> predicates = new ArrayList<>();
        
        if (query.getTitle() != null) {
            predicates.add(cb.like(root.get("title"), "%" + query.getTitle() + "%"));
        }
        
        if (query.getCategoryId() != null) {
            predicates.add(cb.equal(root.get("category").get("categoryId"), query.getCategoryId()));
        }
        
        return cb.and(predicates.toArray(new Predicate[0]));
    };
    
    Page<Book> books = bookRepository.findAll(spec, pageable);
    return books.map(this::toDTO);
}
```

### 4.2 缓存策略

```java
@Service
@CacheConfig(cacheNames = "books")
public class BookCacheService {
    
    // 读取缓存
    @Cacheable(key = "#bookId", unless = "#result == null")
    public BookDTO getBook(Long bookId) {
        return bookRepository.findById(bookId)
            .map(this::toDTO)
            .orElse(null);
    }
    
    // 更新缓存
    @CachePut(key = "#book.bookId")
    public BookDTO updateBook(BookDTO book) {
        Book entity = toEntity(book);
        Book saved = bookRepository.save(entity);
        return toDTO(saved);
    }
    
    // 删除缓存
    @CacheEvict(key = "#bookId")
    public void deleteBook(Long bookId) {
        bookRepository.deleteById(bookId);
    }
    
    // 批量删除缓存
    @Caching(evict = {
        @CacheEvict(key = "#bookId"),
        @CacheEvict(value = "bookList", allEntries = true)
    })
    public void deleteBookAndClearList(Long bookId) {
        bookRepository.deleteById(bookId);
    }
    
    // 条件缓存
    @Cacheable(key = "#bookId", condition = "#bookId > 1000")
    public BookDTO getBookWithCondition(Long bookId) {
        return bookRepository.findById(bookId)
            .map(this::toDTO)
```

```java
    public String maskEmail(String email) {
        if (email == null || email.length() < 3) {
            return "***";
        }
        
        int atIndex = email.indexOf('@');
        if (atIndex <= 0) {
            return "***";
        }
        
        String username = email.substring(0, atIndex);
        String domain = email.substring(atIndex);
        
        if (username.length() <= 2) {
            return "***" + domain;
        }
        
        return username.charAt(0) + "***" + username.charAt(username.length() - 1) + domain;
    }

            return "***";
        }
        
        return phone.substring(0, 3) + "****" + phone.substring(phone.length() - 4);
    }

        if (cardNumber == null || cardNumber.length() < 8) {
            return "****";
        }
        
        return "****-****-****-" + cardNumber.substring(cardNumber.length() - 4);
```

```java
            .message(ex.getMessage())
            .details(ex.getDetails())
            .timestamp(LocalDateTime.now())
            .build();
        
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body(response);
    }

    public ResponseEntity<ErrorResponse> handleValidationException(MethodArgumentNotValidException ex) {
        Map<String, String> fieldErrors = ex.getBindingResult()
            .getFieldErrors()
            .stream()
            .collect(Collectors.toMap(
                FieldError::getField,
                FieldError::getDefaultMessage,
                (existing, replacement) -> existing
            ));

            .code(ErrorCode.VALIDATION_ERROR.getCode())
            .message("Validation failed")
            .details(Map.of("fieldErrors", fieldErrors))
            .timestamp(LocalDateTime.now())
            .build();
        
        return ResponseEntity.status(HttpStatus.BAD_REQUEST).body(response);
    }
```

```java
    public Book createBook(CreateBookCommand command) {
        log.info("Creating book with ISBN: {}", command.getIsbn());
        
        try {
            Book book = bookFactory.create(command);
            Book savedBook = bookRepository.save(book);
            
            log.info("Book created successfully: bookId={}, isbn={}", 
                savedBook.getBookId(), savedBook.getIsbn());
            
            // 结构化日志
            log_structured.info(BOOK_CREATED, 
                Map.of(
                    "bookId", savedBook.getBookId(),
                    "isbn", savedBook.getIsbn().getIsbn13(),
                    "title", savedBook.getTitle(),
                    "timestamp", Instant.now()
                ));
            
            return savedBook;
            
        } catch (InvalidISBNException e) {
            log.error("Failed to create book: invalid ISBN {}", command.getIsbn(), e);
            throw e;
        } catch (Exception e) {
            log.error("Unexpected error creating book with ISBN: {}", 
                command.getIsbn(), e);
            throw new ApplicationException(ErrorCode.INTERNAL_ERROR, 
                "Failed to create book", e);
        }
    }
    
    public void deleteBook(Long bookId) {
        log.warn("Deleting book: bookId={}", bookId);
        
        Book book = bookRepository.findById(bookId)
            .orElseThrow(() -> new BookNotFoundException(bookId));
        
        bookRepository.delete(book);
        
        log.warn("Book deleted: bookId={}, isbn={}", 
            bookId, book.getIsbn().getIsbn13());
        
        log_structured.warn(BOOK_DELETED,
            Map.of(
                "bookId", bookId,
                "isbn", book.getIsbn().getIsbn13(),
                "deletedBy", SecurityContextHolder.getContext()
                    .getAuthentication().getName(),
                "timestamp", Instant.now()
            ));
    }
}
```

## 实现状态（2026-05-31 更新）

### 项目中实际采用的最佳实践

#### 1. 领域模型

- **静态工厂方法**: 所有聚合根使用 `create()` 静态方法，不暴露构造函数
- **不可变设计**: 业务方法不返回值，修改内部状态；值对象（Money, Email, PhoneNumber）不可变
- **乐观锁**: 所有聚合根 `@Version` 字段
- **审计字段**: 所有实体 `created_at` + `updated_at`

#### 2. ID 设计

- **共享值对象**: 16 个 ID 类型定义在 library-shared，跨上下文类型安全
- **JPA 映射**: `@EmbeddedId` + 自定义 `AggregateId` 基类

#### 3. 事件驱动

- **双发模式**: 本地 Spring Event + Kafka（可选注入，优雅降级）
- **Consumer 路由**: `@KafkaListener` → `ObjectMapper.readTree()` → switch(eventType) → Handler
- **异步验证**: `await().atMost(10, SECONDS).untilAsserted()` 在 E2E 测试中

#### 4. 异常处理

- **层次结构**: `DomainException` 基类 + 各上下文子类（携带错误码）
- **全局处理**: `@RestControllerAdvice` + `GlobalExceptionHandler`
- **REST 响应**: 统一 `ApiResponse<T>` 包装

#### 5. 测试策略

- **单元测试**: JUnit 5 + Mockito + AssertJ（79 个测试类）
- **BDD 功能测试**: Cucumber 7.15 + Spring（37 features, 36 steps）
- **E2E JUnit5**: SpringBootTest + EmbeddedKafka + Awaitility（9 tests）
- **E2E BDD**: Cucumber + EmbeddedKafka（9 scenarios）
- **测试数据清理**: `@Before` 中按逆 FK 依赖顺序 DELETE 所有表

#### 6. 代码组织

| 规范 | 实现 |
|------|------|
| 包结构 | domain/application/infrastructure/interfaces 四层 |
| 文件大小 | 多数 200-400 行，最大 800 行 |
| 命名 | `{Entity}Repository`, `{Entity}Controller`, `{Event}Handler` |
| 模块隔离 | 每个上下文独立 Maven 模块，仅通过 Kafka 事件通信 |
